DLIVE-SMR 광고 시스템 연동 규격
===================
이 문서는 SMR 광고 시스템과 DLIVE의 광고 시스템 간의 연동에 대한 내용을 다룬다.
SMR과 타 케이블 사이트 광고를 위한 연동 이슈는 이 문서에서는 다루지 않는다.
